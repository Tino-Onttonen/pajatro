## Mobile Input  
What kind of mobile input will our game implement?  
This game will be implementing the "InputEventScreenTouch" class from Godot. The game has many different boxes/buttons that the user can press, and we will utilize this class with the drawn assets.  
## Objects that will be taking input  
The game will have many different clickable objects, and they are listed here:  
- Play-button.  
- Power-button.  
- Power-up buttons.  
- Settings-button.  
- Language button in the settings menu.
